The solar system visualization uses CSS animations and transforms to create realistic planet orbits. Each planet has its own orbit radius and animation duration, creating a dynamic and engaging display. The planets are positioned using CSS custom properties and transform-origin for precise orbital paths.

The login system uses bcrypt for secure password hashing and comparison. The server implements proper routing to ensure the login page is the first page users see. After successful authentication, users are redirected to the main content using JavaScript's fetch API and proper error handling.

The hero section uses a layered parallax effect with stars and twinkling animations. Multiple background layers move at different speeds using CSS animations and z-index positioning, creating a sense of depth and movement as users scroll.

The planet cards are rendered using JavaScript. A `fetch()` request is made to the Express backend (`/data/planets`), which returns a JSON array. The data is then used to dynamically build Bootstrap card elements and insert them into the page.

A Node.js + Express server handles routing, static file serving, and API endpoints. The server implements proper middleware ordering to ensure correct routing, with the login page as the default route. It also serves JSON data for the planet cards and handles user authentication.
